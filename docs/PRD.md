## 0. Executive Summary

**한 줄 포지션:** 한국 사회과학 연구자를 위한 최초의 논문 수준 통합 분석 플랫폼.  
연구 설계부터 설문 배포, SEM 분석, 한국어 자동 해석, APA 보고서까지 하나의 워크플로우로.

## 1. 서비스 개요

### 1.1 서비스 목적

사회과학 분야 비전문가 연구자도 논문 심사를 통과할 수 있는 수준의 통계 분석을 완성할 수 있도록 지원한다.  
연구 설계 → 설문 제작 → 데이터 수집 → 분석 → 보고서의 전 과정을 단일 플랫폼에서 처리한다.

### 1.2 핵심 가치 제안

| 가치 | 기존 문제 | H.Research 해결 |
|------|----------|----------------|
| 워크플로우 통합 | 5개+ 도구 전환, 매 단계 파일 변환 | 연구 설계 → 보고서 단일 플랫폼 |
| 분석 접근성 | SPSS+AMOS 연 200~400만 원 | 웹 브라우저만으로 즉시, 1/10 가격 |
| 결과 해석 | 통계 출력을 한국어 학술 문체로 직접 작성 | 논문 수준 한국어 해석 자동 생성 |
| 분석 깊이 | SEM·매개/조절 위해 별도 도구 구매 | CFA·SEM·PLS-SEM·PROCESS 통합 |
| 모델 중심 UX | 분석 메뉴를 하나씩 선택해서 실행 | 연구 모델을 그리면 파이프라인 자동 결정 |
| 연구 설계 연속성 | 설문 설계와 분석 변수가 단절 | 설문 제작 시 잠재변수 사전 설정 → 분석 자동 연결 |
| 협업 | 카카오톡·이메일로 파일 주고받기 | 랩 워크스페이스에서 코멘트·승인 일괄 처리 |

### 1.3 타겟 사용자

| 사용자군 | 규모 (국내) | 핵심 니즈 |
|---------|-----------|---------|
| 사회과학 석·박사 재학생 | ~82,000명 | 저비용 + 논문 통과 품질 |
| 사회과학 전임교수 | ~18,000명 | 효율성 + KCI 수준 품질 |
| 연구소 연구원 | ~25,000명 | 빠른 결과 + 협업 |
| 학부생 (진입 채널) | 수십만 명 | 무료 + 사용성 |

## 3. 설문 모듈 (P1)

### 3.1 잠재변수 설정 화면

설문 제작의 첫 번째 단계. 분석할 변수 구조를 먼저 정의한다.

- 잠재변수 이름 생성 → 문항 작성 → 역문항 토글
- 척도 유형: 리커트 5점·7점·명목·서열
- 검증 척도 라이브러리: 표준화 척도 선택 시 문항·역문항 자동 완성
- 역문항 → 분석 시 자동 역코딩 반영

### 3.2 표본 크기 계산 (P1)

설문 배포 전 필요 표본 수 자동 계산. G*Power 대체.

| 분석 유형 | 계산 기준 |
|---------|---------|
| SEM | 잠재변수 수·문항 수 기반 최소 표본 권장 |
| 회귀분석 | 예측변수 수·효과크기·검정력 기반 |
| t검정·ANOVA | 집단 수·효과크기·검정력 기반 |
| 상관분석 | 기대 상관계수·검정력 기반 |

### 3.3 설문 구성 및 배포

| 기능 | 우선순위 |
|------|---------|
| 문항 순서 드래그앤드롭 | P1 |
| 페이지 분리 | P1 |
| 인트로·아웃트로 페이지 | P1 |
| 응답자 화면 미리보기 | P1 |
| 링크·QR 코드 생성 | P1 |
| 응답 기간·최대 응답 수 설정 | P1 |
| 중복 응답 방지 | P1 |
| 조건부 분기 | P2 |
| 미디어 삽입 (사진·영상) | P3 |

### 3.4 응답자 UX 원칙

- 모바일 최적화: 리커트 큰 버튼 터치 응답
- 진행 현황 표시: 페이지 수·진행 바
- 자동 저장: 중간 이탈 후 복귀 시 응답 유지
- 간결한 UI: 문항 집중 화면

### 4.1 응답 현황 패널

- 접속 수 / 진행 중 / 완료 수 / 중도 이탈 수·비율

- 잠재변수 합산 평균 자동 계산
- 데이터 품질 자동 체크: 결측치·정규성·분산 이상 경고
- 개별 응답 로우 데이터 조회

## 5. 모델 캔버스 기능 스펙

### 5.1 화면 구조

| 영역 | 위치 | 역할 |
|------|------|------|
| 문항 패널 | 좌측 | 전체 문항 목록 및 할당 상태 |
| 모델 캔버스 | 중앙 | 잠재변수 노드 및 경로 배치·연결 |
| 사이드 패널 | 우측 | 선택 요소 통계 미리보기 및 분석 결과 |
| 실행 바 | 하단 | 분석 실행·모델 구조 인식·상태 |

### 5.2 노드 종류

**잠재변수 노드 (타원형)**
- 내부에 할당 문항·α·AVE 표시
- 역할별 색상: 독립(파랑) / 종속(초록) / 매개(노랑) / 조절(보라)

**관측변수 노드 (사각형)**
- 단일 문항·인구통계 변수 직접 사용

**생성 방법:**
- A: 문항 다중 선택 후 캔버스 드롭
- B (기본 권장): 캔버스 더블클릭 → 박스 생성 → 문항 드롭

### 5.3 모델 구조 자동 인식

| 캔버스 구조 | 자동 인식 파이프라인 |
|-----------|-------------------|
| 잠재변수 + 경로 없음 | CFA만 실행 |
| 잠재변수 + 단방향 경로 | CFA → CB-SEM |
| 잠재변수 + 조절 경로 | CFA → SEM + 조절효과 |
| 잠재변수 + 매개 경로 | CFA → SEM + 매개(부트스트랩) |
| 관측변수 + 매개 + 조절 | PROCESS 모델 자동 선택 |
| 다층 모드 활성화 | rwg·ICC → HLM |

### 5.4 1단계: 측정모형

**공통방법편의 검증 (P1):**

| 방법 | 기준 |
|------|------|
| Harman's single factor test | 단일 요인 설명 분산 < 50% |
| Marker variable technique | 마커 변수 설정 시 자동 CMV 통제 |
| CFA 기반 CMV 검증 | 방법 요인 추가 모형 비교 |

**타당성 분석:** Cronbach's α · McDonald's ω · EFA · CFA · AVE · CR · HTMT · Fornell-Larcker

**결과 처리:**
- 노드 내 α·AVE 즉시 표시
- 기준 미달 문항 ⚠ + 제거 권장
- 원클릭 제거 후 재실행
- 전체 기준 충족 시 [2단계로 →] 활성화

### 5.5 2단계: 구조모형

**단일 수준 (기본):**
- 경로 연결 → 분석 실행
- 유의성 신호등: 🟢 p<.001 / 🟡 p<.05 / 🔴 p≥.05
- 종속변수 노드 내 R² 표시

**다층 모드 (토글):**

캔버스 우측 상단 토글 활성화 시 레이어 분리. 논문의 다층모형 표현 방식과 동일한 구조.

```
── Level 2 (팀·조직 수준) ──────────
  [리더십] (rwg·ICC 검증 필요)
        ↓
── Level 1 (개인 수준) ──────────────
  [직무만족] ──→ [조직몰입]
```

다층 분석 파이프라인:

| 단계 | 분석 | 기준 |
|------|------|------|
| 1 | rwg · rwg(j) | ≥ .70 권장 |
| 2 | ICC(1) · ICC(2) | ICC(1)>.05, ICC(2)>.50 |
| 3 | Aggregation 적합성 자동 판단 | 한국어 해석 포함 |
| 4 | HLM 실행 | 집단 평균 중심화 자동 |

### 5.6 사이드 패널

| 선택 상태 | 표시 내용 |
|---------|---------|
| 선택 없음 | 전체 데이터 기술통계·분석 전 체크 |
| 잠재변수 노드 | 할당 문항·α 미리보기 |
| 단일 문항 | 평균·SD·왜도·첨도·히스토그램 |
| 경로(화살표) | 두 변수 간 상관계수 |
| 변수 2개 동시 선택 | 산점도 + 상관행렬 |
| 분석 완료 후 | 모형적합도·한국어 해석 전문 |

## 6. 버전 관리 및 분석 옵션

### 6.1 버전 vs 세부값

| 구분 | 저장 시점 | 용도 |
|------|---------|------|
| 버전 | 모델 구조(경로·변수) 변경 시 | 다른 연구 모델 비교 |
| 세부값 | 같은 모델에서 옵션만 변경 시 | 분석 옵션 간 결과 비교 |

```
모델 v1 "초기 모델"
├── 기본 실행    ML · Bootstrap 5,000
├── 세부값 #1   WLSMV · Bootstrap 5,000
└── 세부값 #2   WLSMV · Bootstrap 10,000

모델 v2 "매개변수 추가"
├── 기본 실행    ML · Bootstrap 5,000
└── 세부값 #1   MLR · Bootstrap 5,000
```

> 버전·세부값 개수 제한: 추후 사용 패턴 확인 후 결정.

### 6.2 분석 옵션 패널

**기본 노출 옵션:**

| 분석 | 옵션 | 기본값 | 근거 |
|------|------|--------|------|
| CFA·SEM | 추정 방법 | ML | 연속형 데이터 일반 관행 |
| 부트스트랩 | 반복 수 | 5,000회 | KCI 논문 관행 |
| 부트스트랩 | 신뢰구간 | 95% | 사회과학 표준 |
| 결측치 | 처리 방법 | FIML | SEM 권장 방식 |
| EFA | 회전 방식 | Varimax | 직교 회전 기본 |
| 회귀 | 투입 방식 | 강제 투입 | 가설 검증 기본 |

**고급 옵션 (접어두기):** 표준화 방식 · 수렴 기준 · seed 고정 · 신뢰구간 방식(percentile/BCa)

### 6.3 옵션 설명 방식

ⓘ 클릭 시 세 가지를 함께 표시:
1. 이 옵션이 하는 일 (개념 설명)
2. 다른 옵션으로 바꾸면? (선택지별 영향)
3. 내 데이터에는? (현재 데이터 특성 기반 자동 권장)

## 7. 조직 관리 및 랩 협업

### 7.1 조직 구조

```
조직 (대학·연구소·기업)
├── 조직 관리자
│     └── 진행 상황 모니터링만 가능
│         (데이터·모델·결과 접근 불가)
└── 멤버 (학생·연구원·교수)
      └── 본인 프로젝트 완전 독립
```

> MVP: 2계층(관리자·멤버). 대학 플랜: 3계층(조직·학과·멤버) 추가.

### 7.2 조직 관리자 대시보드

**볼 수 있는 것:**

| 항목 | 상세 |
|------|------|
| 멤버 현황 | 전체·활성·미사용 멤버 수 |
| 사용 현황 | 멤버별 마지막 접속일·분석 실행 횟수·완료 건수 |
| 프로젝트 현황 | 진행 중·완료·미시작 수 (제목만, 내용 불가) |
| 설문 현황 | 배포 중인 설문 수·응답 수 (내용 불가) |
| 라이선스 현황 | 사용 시트 수·만료일·사용률 |

**볼 수 없는 것:** 데이터셋 내용 · 분석 모델·결과 · 설문 문항·응답 · 보고서 내용

### 7.3 랩 워크스페이스 (P1)

**권한 구조:**

| 역할 | 접근 범위 |
|------|---------|
| 지도교수 | 학생이 공유한 프로젝트 전체 |
| 학생 | 본인 프로젝트만 (공유 여부 직접 결정) |
| 조직 관리자 | 진행 상황만 (랩 내용 접근 불가) |

**검토 요청 플로우:**

```
학생: 분석 완료 → [검토 요청] → 요청 메시지 → 교수 알림
      ↓
교수: 결과 확인 → 캔버스·테이블·해석에 코멘트
      ↓
학생: 코멘트 확인 → 수정 → 새 버전 저장 → 재요청
      ↓
교수: [이 버전 승인 ✅] → 승인 뱃지 → 학생 알림
```

**코멘트 가능 위치:**
- 캔버스 특정 경로·노드
- 결과 테이블 특정 수치
- 한국어 해석 특정 문장
- 모형적합도 수치

## 8. 통계 분석 모듈

### 8.1 기초 분석 (P0)

| 분석 기법 | 포함 항목 | 출력 |
|---------|---------|------|
| 기술통계 | 빈도·% · 평균·SD·왜도·첨도 | APA 인구통계 특성표 자동 |
| t검정 | 독립·대응·단일표본 | Cohen's d · 95% CI |
| ANOVA | 일원·이원, 사후검정 | η² 효과크기 |
| 비모수 검정 | Mann-Whitney · Kruskal-Wallis | 순위 기반 효과크기 |
| 교차분석 | χ² · 기대빈도 | Cramer's V · Phi |
| 상관분석 | Pearson · Spearman · 편상관 | 상관 매트릭스 히트맵 |

### 8.2 측정모형 분석 (P0)

| 분석 기법 | 포함 항목 | 비고 |
|---------|---------|------|
| 신뢰도 | Cronbach's α · McDonald's ω | 문항 제거 시 α 변화 자동 |
| EFA | 주축 요인법 · 직교·사교 회전 · 패럴렐 분석 | |
| CFA | lavaan · ML / WLSMV / MLR | 이분형·서열형 데이터 지원 |
| ESEM | EFA + CFA 통합 | P2 |
| 수렴·판별타당도 | AVE · CR · HTMT · Fornell-Larcker | |
| 공통방법편의 | Harman · Marker · CFA 기반 | P1, KCI 심사 대응 |

### 8.3 구조모형 분석 (P0)

| 분석 기법 | 포함 항목 |
|---------|---------|
| 다중 회귀 | 위계적·강제 투입 · VIF · Cook's Distance |
| 매개분석 | 부트스트랩 5,000회 · 95% CI · FIML |
| 조절분석 | 상호작용항 자동 · 평균중심화 · Johnson-Neyman |
| 조절된 매개 | PROCESS Model GUI화 |
| CB-SEM | lavaan · 측정·구조모형 분리 |
| PLS-SEM | plspm/cSEM · HTMT · 부트스트랩 |
| 다중집단 분석 | 형태·측정·구조 동일성 · Δχ²·ΔCFI |

### 8.4 집단 수준 분석 (P1)

| 분석 기법 | 포함 항목 |
|---------|---------|
| rwg · rwg(j) | 집단 내 합의도 |

| HLM / 다층모형 | 개인-집단 위계 구조 (P2) |

### 8.5 데이터 품질 (P1)

|------|------|

## 9. 자동 해석 엔진 (P0)

| 수준 | 데이터인 | H.Research 목표 |
|------|---------|----------------|
| 결과 서술 | "유의한 차이가 있다(p<.05)" | β·t·p·95% CI 논문 직접 기입 수준 |
| 효과크기 | 미포함 | Cohen's d·η²·f² + "중간 효과크기" 언어 서술 |
| 모형적합도 | 미지원 | "RMSEA=.05, CFI=.96으로 양호" 자동 문장 |
| 매개효과 | 미지원 | "간접효과 β=.14, 95% CI [.07, .23] 완전매개 확인" |
| 공통방법편의 | 미지원 | "단일 요인 설명 분산 23.4%로 CMV 우려 낮음" |
| rwg·ICC | 미지원 | "rwg=.87, ICC(2)=.71로 집단 수준 aggregation 적합" |

> 구현: 규칙 기반 템플릿 엔진 + Claude API. v1.0 한국어 전용 → v2.0 영어 추가.
